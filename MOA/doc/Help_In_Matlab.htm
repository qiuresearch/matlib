Getting Help in Matlab

If you are new to Matlab, on firing it up type "helpdesk" at the command prompt. This will bring up a browser with just a little of Matlab’s documentation. The "Using Matlab" option is a good introduction to the elementary features of Matlab. The manuals are also online which are always good for a read. In particular, don’t neglect the amazing power of Matlab through pre-existing functions, toolboxs, scripting and the ability to build interpreted functions online.

However, if you know the general area you are interested in (eg. "graphing" or "file i/o") but just don’t know your options or can’t remember the name of the desired function, then the command, "helpwin", may be the way to go. This spawns a set of hierarchal windows listing functions in relatively logical groups.

Alternatively, if you remember the command, or an associated command, typing,

help "function_name"

will produce a handy little memory prompt. This feature is available for all the functions in this library, although the help can be a trifle obscure.
