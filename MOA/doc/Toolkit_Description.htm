Toolkit Description


Moa is a MATLAB/C toolkit for analysing X-ray images gathered on the Gruner-Tate CCD detectors using the TV6 data acquisition package. Each such image consists of a large (typically 512*512 pixel) image saved in a non-standard tiff format. MOA was developed to handle particulate SAXS images but many of the tools have wider use. The tools of MOA consist of functions which may be used at the Matlab command line, within Matlab scripts, functions and also within C-programs.

MOA allows you to -     Load images in Matlab as arrays.

                                      Load the ATOD tag into Matlab as an array.

                                      Remove stray radioactive events (zingers) from
                                images by statistical dezingering and correct images for flat- field intensity and distortion.

                                      Perform subtraction, multiplication, division, averaging or, add or any other matrix operation on images.

                                      Normalize images with respect to incident X- ray flux.

                                      Display images and select a region of interest for display, integration or statistical purposes.

                                      Calibrate X-ray scattering from a Silver stearate or similar laminar calibrant.

                                      Perform radially averaged integrals of Intensity versus scattering angle.

                                      Display Intensity, Guinier and Kratky plots.
